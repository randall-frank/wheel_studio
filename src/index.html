About RC Wheel Model Studio

RC Wheel Model Studio lets one explore and generate RC wheel STL files from configurable OpenSCAD model.

Use the parameter panel to tune the wheel and generate a 3D preview before downloading the STL. The Save Params and Load Params buttons can be used to save and load the current parameter settings.
